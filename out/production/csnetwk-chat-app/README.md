# csnetwk-chat-app

## Set up (Windows)
```
1. Navigate to folder.
2. Run the server in cmd: 
    ` run runServer `
3. Run clients in separate cmd windows:
    ` run runClient `
```

## Set up (MacOS)
```
1. Navigate to folder.
2. Run the server in the terminal: 
    ` sh runServer.sh `
3. Run clients in separate terminal windows:
    ` sh runClient.sh `
